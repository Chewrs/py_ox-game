# tic tac toe
this is the simple tic tac toe game writen in python
## installation
1. Open the Terminal 

2. clone this project

```
git clone https://github.com/Chewrs/py_ox-game.git
 ```

## Runing
Go to the project

```
cd py_ox-game
```

Run the game

```
python3 ox.py
```

## Note
> this game only work with Mac os !!!
